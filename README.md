# Personality Drink Test————基于cursor快速实现

## 项目简介
趣味性格测试，通过回答问题生成独特的饮品类型。

## 功能特性
- 响应式设计
- 动态结果生成
- 图片保存
- 动画效果
- 分享功能

## 技术栈
- HTML5
- CSS3
- JavaScript
- html2canvas

## 使用说明
1. 访问页面
2. 回答问题
3. 查看结果
4. 保存/分享

## 开发指南
```bash
git clone https://github.com/nia717/personality-drink-test.git
cd personality-drink-test
```
